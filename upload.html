How to Add Music to the Flow Library

To add new albums or tracks to Flow, follow these steps:

  1. In your project folder, go to: src/data/album/
  2. Create a new folder using your album name:
    src/data/album/your-album-name/
  3. Inside this folder, place all your audio tracks (.mp3, .wav, etc.) and a cover image (e.g. cover.png).
  4. Create an infos.json file in that same folder, describing the album and its tracks. Example: 
    {
  "title": "Album Title",
  "artist": "Artist Name",
  "cover": "cover.png",
  "tracks": [
    {
      "title": "Track One",
      "file": "track1.mp3",
      "duration": "3:21"
    },
    {
      "title": "Track Two",
      "file": "track2.mp3",
      "duration": "4:05"
    }
  ]
}
  5. Then, open the file: src/data/album.json
    and add the name of your album folder to the array. Example: 
    [
  "chillwave-vol1",
  "ambient-beats",
  "your-album-name" // <-- Add this line
]
  6. Save all files and refresh the Flow player. Your new music should now be listed in the library!

Need help or found a bug? You can open an issue or contribute via the project’s GitHub.
