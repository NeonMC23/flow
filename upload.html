Keep the library tidy with a simple album-based structure.

How to add music

  1. Go to src/data/album/.
  2. Create a folder for your album: src/data/album/your-album-name/.
  3. Add audio files and a cover image (for example cover.png).
  4. Create an infos.json file in that folder with this structure:
{
  "title": "Album Title",
  "artist": "Artist Name",
  "cover": "cover.png",
  "tracks": [
    {
      "title": "Track One",
      "file": "track1.mp3",
      "duration": "3:21"
    },
    {
      "title": "Track Two",
      "file": "track2.mp3",
      "duration": "4:05"
    }
  ]
}
  1. Add your album folder name in src/data/album.json:
[
  "chillwave-vol1",
  "ambient-beats",
  "your-album-name"
]

Refresh the player and your album will appear in the library. If you want to share fixes or improvements, open a pull request on GitHub.
